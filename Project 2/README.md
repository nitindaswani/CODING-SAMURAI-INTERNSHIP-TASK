# Project 2: To-Do CLI App

## Overview

A command-line to-do list application that stores tasks in a local text file (`tasks.txt`).

## Features

- View all tasks
- Add new task
- Update existing task
- Delete task
- Persistent storage using `tasks.txt`

## Requirements

- Python 3.x

## Run

```bash
python "to do app.py"
```

## Menu Options

1. View Tasks
2. Add Task
3. Update Task
4. Delete Task
5. Exit

## Data File

- Task data is saved in `tasks.txt` in the same folder.
