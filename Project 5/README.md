# Project 5: Web Scraping with BeautifulSoup

## Features

- Fetches a public webpage using `requests`
- Parses HTML with `BeautifulSoup`
- Extracts structured data (title + link)
- Saves output to `scraped_data.csv`
- Handles request and file writing errors

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python "web scraping app.py"
```

## Notes

- Scrape only publicly accessible pages.
- Follow website terms of service and robots policy.
