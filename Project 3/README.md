# Project 3: Weather Application Using API

## Features

- Fetches live weather data using OpenWeatherMap API
- Supports Celsius and Fahrenheit units
- Handles invalid city names, API key errors, and network issues
- Allows repeated weather checks in a loop

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Get an API key from OpenWeatherMap.
3. Option A: Set environment variable `OPENWEATHER_API_KEY`.
4. Option B: Enter API key when prompted at runtime.

## Run

```bash
python "weather app.py"
```
