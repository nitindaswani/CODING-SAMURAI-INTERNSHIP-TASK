# Project 1: Calculator (Tkinter)

## Overview

A basic GUI calculator built using Python Tkinter. It supports common arithmetic operations and clear functionality.

## Features

- Graphical interface with buttons and display
- Supports `+`, `-`, `*`, `/`
- Clear button (`C`)
- Evaluates expressions when `=` is clicked
- Handles invalid expressions by showing `Error`

## Requirements

- Python 3.x
- Tkinter (usually included with standard Python installation)

## Run

```bash
python calculator.py
```

## Usage

- Click number and operator buttons to form an expression.
- Click `=` to evaluate.
- Click `C` to clear the display.
